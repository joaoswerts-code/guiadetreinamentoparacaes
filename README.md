# Guia de Treinamento para Cães em 7 Dias

Landing page de vendas com quiz interativo e VSL.

## Como hospedar no GitHub Pages

1. Faça upload de **todos** os arquivos desta pasta para o seu repositório no GitHub.
2. No repositório, vá em **Settings > Pages**.
3. Em "Source", selecione a branch onde fez o upload e a pasta raiz `/` (ou `/docs` se preferir).
4. Salve. O site ficará disponível em alguns minutos.

## Estrutura dos arquivos

```
index.html         ← Página principal
favicon.svg        ← Ícone do site
assets/
  vsl.mp4          ← Vídeo da VSL
  index-*.js       ← JavaScript da aplicação
  index-*.css      ← Estilos da aplicação
```
